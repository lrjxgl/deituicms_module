# deituicms_module
 DeituiCMS的应用模块插件,通过模块插件轻松搭建自己想要的网站        
 DeituiCMS:https://github.com/lrjxgl/deituiCMS ,官网：https://www.deituicms.com   
 要使用这些模块需要先安装deituicms，然后将它们复制到module目录，然后在后台插件管理安装。
